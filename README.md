# 🌍 ClimAct | SDG 13: Climate Action Website

**ClimAct** is an interactive, educational platform dedicated to **Sustainable Development Goal 13: Climate Action**. This project was collaboratively developed by a team of four students to inspire environmental change through interactive tools, scientific education, and community engagement.

---

## 📂 Project Repository Structure

The website is built on a shared design system with specific contributions categorized by student roles:

### 🏠 Core Pages
* **`home.html`**: The central entry point introducing the site's mission and focus areas.
* **`splash.html`**: An introductory presentation page with an automatic redirect to the Home page.
* **`sitemap.html`**: A comprehensive navigation directory for all pages on the site.

### ⚡ Interactive Tools & Community
* **`ais.html`**: **Action Impact Simulator** – An interactive tool to calculate the carbon impact of daily choices.
* **`gallery.html`**: A dynamic photo gallery showcasing climate challenges with user customization features.
* **`feedback.html`**: A community programme directory and feedback submission form with live validation.
* **`profile.html`**: User profile management and personalization page.
* **`team.html`**: Introductions to the student developers behind the project.

## ✨ Key Features

### 1. Action Impact Simulator (`ais.html`)
Users select from 8 climate actions to calculate a total impact score.
* **Dynamic Visuals**: The feedback panel and page background transition between colors (Red, Orange, Green) based on the calculated impact level.
* **Live Feedback**: Immediate textual and numeric scoring based on user selections.

### 2. Interactive Media Gallery (`gallery.html`)
A JavaScript-driven grid that allows users to explore environmental photography.
* **Customization**: Users can personalize the viewing experience by changing the modal background color, text color, and font style in real-time.

### 3. Feedback System (`feedback.html`)
A robust engagement tool featuring:
* **Programme Directory**: A scrollable list of local initiatives like the "Zero Waste Challenge".
* **Form Validation**: Real-time character counting (500-character limit) and comprehensive field error-checking.

---

## 🛠️ Technical Specifications

* **Global Design System (`style.css`)**: Implements a unified architecture using CSS variables for colors, spacing, and shadows to ensure a cohesive look.
* **Responsive Design**: Built using CSS Grid and Flexbox with specific breakpoints for mobile, tablet, and desktop viewports.
* **Accessibility**: Features semantic HTML5 elements, ARIA labels for interactive components, and keyboard navigation support.
