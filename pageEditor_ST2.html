Splash Screen (splash.html)

Description

The Splash Screen serves as the introductory presentation page for the ClimAct website. It features a full-screen background image, the ClimAct logo, a brief welcome message, and an automatic redirect to the Home Page. The splash screen provides an engaging first impression while the site content is presented.

Technical Details

Auto-Redirect: Uses a meta refresh tag with a 10-second delay to automatically redirect users to the Home Page. This satisfies the requirement for automatic redirection without relying on JavaScript.

CSS Animation: Features a custom CSS loading spinner animation using keyframes and border-radius manipulation. The loader provides visual feedback indicating that the site is preparing to load.

JavaScript Countdown: A real-time countdown timer displays the seconds remaining before redirect. The timer updates every second using setInterval and automatically redirects when reaching zero.

Skip Button: A clearly labelled "Skip to Home Page" button allows users to bypass the countdown and navigate immediately. This follows accessibility best practices by not forcing users to wait.

SDG Action Impact Simulator (ais.html)

Description

The Action Impact Simulator is an interactive tool that allows users to explore how different everyday actions contribute to climate action. Users select from 8 action cards (such as reducing meat consumption, using public transport, planting trees) and receive immediate visual feedback on their collective impact score.

Technical Details

Action Cards: Eight clickable cards, each representing a different sustainability action with an icon, title, description, and impact level badge (Low, Medium, or High). Cards toggle between selected and unselected states with CSS class changes.

Scoring System: Each action has a numeric impact value. JavaScript calculates the total score of selected actions and maps it to one of four feedback levels: No Actions (0), Low Impact (1-20), Medium Impact (21-40), and High Impact (41+). The background colour transitions dynamically based on the current impact level.

Dynamic Feedback: A results panel displays the current score, impact level badge, descriptive feedback text, and a count of selected actions. A progress bar visually represents the impact level.

Reset Functionality: A reset button clears all selections, resets the score to zero, restores the original background colour, and clears all feedback — returning the simulator to its initial state.

Content Page (content_ST2.html)

Description

The content page covers Sustainable Transport and explores how greener travel choices can reduce carbon emissions. It includes five sections covering the transport problem, cycling and walking, public transport, electric vehicles, and practical student travel tips.

Technical Details

Internal Navigation: A sidebar navigation component provides anchor links to each of the five main sections, enabling quick jumps within the document.

Visual Content: Each section includes a high-quality photograph sourced from Unsplash with descriptive alt text for accessibility compliance.

Go to Top Button: A fixed-position button in the bottom-right corner links back to the top of the page using an anchor reference.
