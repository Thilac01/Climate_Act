Shared HTML Template & Global CSS

Description

The Shared Template is the foundation used across all website pages. It includes the consistent header with the ClimAct logo, navigation bar with responsive mobile menu, and footer with quick links, contact sections, and page information. The global CSS file (style.css) implements a complete design system with CSS custom properties for colours, typography, spacing, shadows, and responsive breakpoints.

Technical Details

Header: The header features a sticky navigation bar with a custom SVG logo, site title and tagline, and a hamburger menu button for mobile viewports. The mobile menu uses JavaScript to toggle a CSS class that reveals the nav links in a vertical dropdown.

Footer: The footer uses a CSS Grid layout with four equal-width sections. On mobile (below 768px), it collapses to a single-column layout. Each section contains a heading and a list of links or information.

CSS Architecture: The stylesheet uses CSS custom properties (variables) for all design tokens including colours, spacing, border-radius values, shadows, and font stacks. This ensures visual consistency and easy theme modifications. Responsive design uses three breakpoints: 992px, 768px, and 480px.

Home Page (home.html)

Description

The Home Page serves as the main entry point of the website. It features a full-width hero section with a background image, headline, and call-to-action button. Below the hero are three mission cards highlighting key aspects of SDG 13. An interactive content gallery section displays four click-able cards, one for each student's content page, with images and descriptive text.

Technical Details

Hero Section: Uses a CSS background image with overlay gradient, centred text, and responsive sizing. The hero adapts to all screen sizes using relative units and flexible padding.

Mission Cards: Implemented using CSS Grid with auto-fill and minmax for responsive columns. Each card includes hover effects (translateY and box-shadow transitions) for visual feedback. Cards display an emoji icon, heading, and paragraph.

Content Gallery: Four cards linking to individual content pages. Each card has a large background image, an overlay with text, and smooth hover transitions for scaling and opacity changes.

Gallery Page (gallery.html)

Description

The Gallery page features a dynamic, JavaScript-driven image gallery with climate action photographs. Users can click thumbnails to open an enlarged modal view with title, description, and source attribution. User customisation controls allow visitors to change the page background colour, text colour, and font family to suit their preferences.

Technical Details

Thumbnail Grid: Uses CSS Grid with auto-fill columns. All image data (source, title, description, alt text) is stored in a JavaScript array and rendered dynamically using DOM manipulation. This avoids hardcoding image elements in HTML.

Modal System: A full-screen modal overlay opens on thumbnail click. The modal displays the full-size image, title, caption, and photographer attribution. It closes via a close button, clicking outside the modal, or pressing the Escape key.

Customisation Controls: Three input fields (background colour picker, text colour picker, and font family dropdown) allow users to personalise the page appearance in real-time using JavaScript event listeners that modify the document body's inline styles.

Content Page (content_ST1.html)

Description

The content page covers Renewable Energy Solutions as a key topic for climate action. It includes five detailed sections: Solar Energy, Wind Power, Hydroelectric Energy, Benefits of Renewables, and What Students Can Do. Each section features high-quality images with descriptive alt text.

Technical Details

Internal Navigation: A navigation sidebar at the top provides anchor links to each section, allowing quick jumps to specific content areas. This uses standard HTML anchor references (href="#id").

Go to Top Button: A fixed-position circular button with an arrow icon appears in the bottom-right corner, allowing users to instantly scroll back to the top of the page.

Content Structure: Uses semantic HTML with proper heading hierarchy (h1 for page title, h2 for section headings, h3 for sub-sections). All images include descriptive alt attributes for accessibility.
